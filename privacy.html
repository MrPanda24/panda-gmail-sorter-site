Privacy Policy

Panda Gmail Sorter

This application is a personal automation tool used only by its developer and owner to organize their own Gmail inbox. It is not distributed to, or used by, any other individuals.

Data accessed

Email metadata (sender, subject, and a short snippet) from the developer's own Gmail account, accessed via the official Gmail API using OAuth 2.0. Full email bodies are never accessed.

How data is used

Each email's metadata is sent to the Anthropic Claude API for one-time classification (Important / Business / Spam) and a corresponding Gmail label is applied to that message. No email content is stored, sold, or shared with any third party beyond this single classification call.

Data retention

No email data is permanently stored by this application outside of Gmail itself. A local sync-state file is kept only to avoid re-processing the same messages, and contains no email content.
